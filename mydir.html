Celik Minda

This course provides students with the knowledge and hands-on practice they need to produce and deliver high-quality animations, interfaces, games and applications specifically designed to take advantage of mobile device capabilities. The course allows a developer with no Flash experience to quickly start improving their mobile workflow using for example jQuery Mobile and iPhone SDK, and to develop applications much faster than with other technologies. The course is task based, with students learning by actually building these applications. Developers will learn how to create applications for existing mobile platforms for example iPhone and Symbian platforms. The course focuses on mobile best practices and students will learn how to build applications that run efficiently on lower powered mobile devices
